## Hotel Guest Feedback Form
A responsive, user-centric feedback component designed to capture guest experiences and improve service quality. This project demonstrates clean HTML5 structure, accessible form design, and client-side validation.

### Key Features
Intuitive Rating System: Uses radio buttons or star-rating logic for service evaluation.

Categorized Feedback: Specific sections for Housekeeping, Dining, and Front Desk experiences.

Responsive Layout: Fully optimized for mobile devices, allowing guests to provide feedback on the go.

Accessibility (A11y): Built with semantic HTML and proper <label> tagging for screen readers.
